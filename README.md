# HR Attrition Dashboard

## Overview
The HR Attrition Dashboard is designed to provide insights into employee attrition rates using visual representations. It aims to help the HR department make data-driven decisions regarding employee retention strategies.

## Features
- **Interactive Visualizations**: Users can explore various metrics related to employee attrition.
- **Data Filtering**: Filter data by department, job role, and tenure.
- **Real-time Updates**: The dashboard updates dynamically with new data inputs.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shivamnaryal09-DS/HR-ATTRITION-DASHBOARD.git
   ```
2. Navigate into the directory:
   ```bash
   cd HR-ATTRITION-DASHBOARD
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To launch the dashboard, run:
```bash
streamlit run app.py
```
